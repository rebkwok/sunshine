Website Administration Pages - Session Types

From the Admin Dashboard, Instructors can be added or edited.

IMPORTANT: if you delete a session type, all sessions associated with that session type will also be deleted! (You will see a warning page first.)

Session Types summary

The Session Types summary page shows the session types that have been entered. To edit an existing session type, click on its name. To add a new one, use the Add Session Type button.

If "Display Session" is ticked, this session type will be listed on the website homepage in the links under "Classes", and the link will go to a separate page for that session type, displaying the class, description, and photo (if available).

In the example below, "Pole Fitness and Tricks" is a separate session type, but doesn't need a separate class description page to "PoleFit", so it is not ticked as "display session".

A photo can optionally be uploaded for each session type. If a photo is uploaded, it will appear on the right hand side of the class description webpage. If no photo is uploaded, the description text will just be the width of the page.


Editing a session description

Click on a session type from the summary page to edit its details. Note that if you want line breaks between paragraphs on the description webpages, you must put a double line break in the description (i.e. make sure there is vertical space between paragraphs as in the example below.
