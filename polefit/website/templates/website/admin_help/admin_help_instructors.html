Website Administration Pages - Instructors

From the Admin Dashboard, Instructors can be added or edited.

IMPORTANT: if you delete an instructor, all sessions associated with that instructor will also be deleted! (You will see a warning page first.)

The Instructors summary page shows the instructors that have been entered. To edit an instructor's details, click on their name. To add a new instructor, use the Add Instructor button. If an instructor is a "Regular Instructor", their name, description, and photo (if available) will be shown on the Instructors page of the website. Instructors who are not ticked as "Regular Instructor" can be assigned to individual sessions, but will not be displayed on the main Instructor overview page.


The "photo uploaded" column simply shows whether a photo has been attached for that instructor yet. If no photo is uploaded for a "regular instructor", a placeholder image will be shown on the Instructors webpage.

Photos for instructors must be square! They can be any size, but must be equal height and width otherwise the circular image will be distorted on the webpage.
