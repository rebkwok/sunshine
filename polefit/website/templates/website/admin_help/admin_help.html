Website Administration Pages - Logging in and the Admin Dashboard

To log in to the administration pages, go to the admin site and enter your admin user name and password.


After you log in, you will see the Admin Dashboard. The first two columns contain links to change/add content to the website. The third column contains useful links to actual pages on the site, including this help site. The links in the menu on the left hand side of this page provide further information on editing each of these sections. The "Home" link in the top menu bar will always bring you back to the Admin Dashboard page.
