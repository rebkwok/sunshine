Website Administration Pages - Events

An event is any one-off competition, showcase, or other event that you want to promote on the website. It will appear on the Events webpage. Events do not appear in the Timetable, so if you have a one-off workshop that is a session in the timetable, you need to enter it as an Event as well in order for it to appear on the Events page.

Events are displayed on the Events page ONLY if they are in the future. They are displayed in date order (by the date of the Event) irrespective of the order in which they are entered on the Admin page.


To edit an event, click on its name. To add a new event, use the Add New Event button. A new event entry only requires a name, date and start time.
