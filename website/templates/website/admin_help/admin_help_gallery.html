Website Administration Pages - Gallery

From the Admin Dashboard, use the Gallery links to add new pictures for the Gallery.

Adding pictures by category

The easiest way to add multiple pictures is to upload by category. Create a category (e.g. Pole Classes, Hoop, Showcase 2014, Pole Competition 2013 etc) and use it to categorise the images on the page. The Gallery webpage also has the option to show all pictures as well as by category

Each picture must have a category (but you could set up a Miscellaneous or Other category for anything that isn't easily classified.

To add a category, click on Categories from the Admin Dashboard, and click the Add New Category button (you can also click on an exisiting category to edit it and add additional pictures to it). Give the category a name and use the Choose File buttons to select your pictures. If you want to add more than one image, click "Add another image" to add more rows.


Editing pictures by Image

Using the Categories option is the easiest way to upload pictures. However, you can't change the category that an already-uploaded picture is assigned to. In order to move a picture to another category, use the Images option from the Admin Dashboard. You will see a list of all uplodaed pictures (depending on how many there are, it may be a very long list). You can click on the Category column heading to sort the pictures by Category if you want.


To change a picture's category, find the picture you want to change and click on its name. You can now change its category by selecting a differnt one from the drop down list, or you can add a new category using the "+" button.
