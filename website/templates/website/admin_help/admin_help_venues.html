Website Administration Pages - Venues

Venues are entered on the Admin pages so that a venue can be assigned to each session (class/workshop) and to Events. However, the Venue pages on the website (with the location maps) are not populated from the Admin pages. If you need to add a venue to the website Venues page, please contact Becky.

IMPORTANT: if you delete a venue, all sessions associated with that venue will also be deleted!

A venue entry is only required to have a name; the address and postcode fields are optional. A "Venue TBC" entry can be used as a placeholder if necessary, when a venue is not confirmed yet.
