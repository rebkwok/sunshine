Timetable

Enter scheduled weekly classes to appear on the timetable page.

Setup

Some basic setup is required before you can add classes:

  • Class types: Each class must have a class type; this is just a name (e.g. Pole Fitness, Stretch etc) and an optional description. If you tick the "display session" box, it (name and description) will appear on the home page (name and description). The "index" is used for ordering on the home page.
  • Categories: optionally assigned to a timetable class, and used to colour-code classes on the timetable page, so they can be grouped in some way. You can make a categories that correspond directly to class types, or use them to group class types - e.g. if there are "Pole Dance" and "Pole Fitness" class types, a "Pole" category could be assigned to both, with a colour code. All of these classes will be shown in the category colour on the timetable.
  • Venue: Each class must be assigned a venue. Only the abbreviation is displayed on the timetable/booking pages.

Upload Timetable

Upload timetable sessions as regular classes that will be available to book online. Select required sessions to upload, and start/end dates. Class names are generated from timetable sessions as "Session Name (Session Level)". i.e. a session called "Aerial Hoop" with level "Level 1" will be uploaded with the name "Aerial Hoop (Level 1)".

You have the option to upload classes with or without "show on site". If you untick the box, classes will not be immediately available to book, and you can go to the "Book" tab on the website to review and check they look correct (they will be displayed to logged-in staff users only). You can turn "show on site" on/off for multiple classes at once in the "Classes/Workshops" section of the admin.

IMPORTANT: the system will check for existing classes by name, date and venue to avoid creating duplicates. If you change any of these fields on a timetable session OR class (or if you change level on a timetable session) and then re-upload for the same dates, you will create duplicate classes.

Classes/Workshops

For online class/workshop booking.

Classes/Workshops are only displayed on the site if the "show on site" checkbox has been ticked.

Regular Classes

Generated via the Upload Timetable; can also be added individually.

Classes are available to book online, but do not take payment. Reminders will be sent to students who book more than 6 hours before the class. (Sent up to 48 hrs before the class - depending on when they booked - to give them time to cancel within the 24 hr cancellation period.)

Workshops

One-off classes, workshops or events which are not displayed on the timetable pages as regular sessions.

Generate workshops manually. These are also available to book online and do not take payment.

Cancelling classes/workshops

    There are 2 ways to cancel a class/workshop:
  1. From the class/workshop list, check the box for the class(es) you want to cancel, select "Cancel class..." from the drop down menu, and click Go
  2. From within the page for an indiviudal class/workshop, use the "Cancel class" link in the menu on the right hand side of the page.

Both options will cancel the class and send notification emails to anyone with open bookings.
